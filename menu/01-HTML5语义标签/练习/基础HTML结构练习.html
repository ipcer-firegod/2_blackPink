基础HTML结构练习

练习目标

通过本次练习，你将掌握HTML的基本结构和常用标签的使用方法，创建一个简单但语义化的网页。

练习任务

创建一个介绍你最喜欢的书籍或电影的网页，包含以下元素：

  • 完整的HTML文档结构（DOCTYPE、html、head、body）
  • 在head中设置正确的字符集和视口
  • 设置一个合适的页面标题
  • 使用h1标签作为主标题
  • 使用h2-h4标签作为章节标题
  • 使用p标签添加至少3个段落的内容
  • 使用img标签插入相关图片（可以使用相对路径）
  • 使用ul或ol标签创建一个列表（例如书籍的主要人物或电影的精彩片段）
  • 使用a标签添加至少一个外部链接（例如到豆瓣或IMDb的链接）
  • 使用适当的语义化标签（header、main、footer等）

提示

  • 确保所有HTML标签都正确闭合
  • 为图片添加alt属性以提高可访问性
  • 为链接添加合适的title属性
  • 保持代码整洁，使用适当的缩进

代码示例

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>我最喜欢的书籍</title>
</head>
<body>
    <header>
        <h1>我最喜欢的书籍：《xxx》</h1>
    </header>
    
    <main>
        <section>
            <h2>内容简介</h2>
            <p>这里是书籍的简介内容...</p>
        </section>
        
        <section>
            <h2>主要人物</h2>
            <ul>
                <li>人物1：描述...</li>
                <li>人物2：描述...</li>
            </ul>
        </section>
        
        <section>
            <h2>相关资源</h2>
            <a href="https://example.com" title="访问官网">访问官网</a>
        </section>
    </main>
    
    <footer>
        <p>© 2025 我的练习作品</p>
    </footer>
</body>
</html>

完成后

完成练习后，请：

  1. 在浏览器中打开你的HTML文件，检查是否正确显示
  2. 尝试在代码中添加一些简单的CSS样式，让页面看起来更美观
  3. 思考如何进一步完善这个页面
